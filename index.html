Documentation

This page is for troubleshooting and questions regarding the POS system.

Getting Started

  • Launch ios App on iPad
  • Select Menu Type
  • Select Item
  • Select options/add ons
  • Repeat until order is finished
  • Press "Pay"

Troubleshooting

  1. Printer Error

    Ensure Printer is connected and status light is solid blue.

    Ensure Printer has paper.

    Ensure Printer is connected to the same network.
